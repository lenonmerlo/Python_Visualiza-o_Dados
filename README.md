# Projeto de Visualização de Dados: Gráficos de Composição e Relacionamento

Este projeto foi desenvolvido com o auxílio do professor Afonso da Alura. Nele, exploramos diversas técnicas de visualização de dados para criar gráficos de composição e relacionamento, utilizando bibliotecas populares em Python como Matplotlib e Seaborn. Nosso objetivo foi apresentar insights valiosos de maneira clara e intuitiva.

## Destaques do Projeto

- **Análise de Composição**: Gráficos que mostram a distribuição de categorias dentro de um conjunto de dados.
- **Análise de Relacionamento**: Gráficos que exploram a relação entre duas ou mais variáveis.
- **Bibliotecas Utilizadas**: Matplotlib, Seaborn, Pandas.
- **Metodologia**: Seguindo as melhores práticas de visualização de dados para garantir clareza e precisão nas interpretações.
